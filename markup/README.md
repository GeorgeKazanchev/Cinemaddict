# Директория с исходной вёрсткой

- `css/` &dash; стили;
- `fonts/` &dash; нестандартные шрифты;
- `img/` &dash; графика;

- `list.html` &dash; список фильмов;
- `list-empty.html` &dash; список фильмов при отсутствии фильмов;
- `list-no-extra.html` &dash; список фильмов без блоков &laquo;Top rated&raquo; и &laquo;Most commented&raquo;;
- `list-filter.html` &dash; список фильмов с применённым фильтром;
- `list-sort` &dash; список фильмов с применённой сортировкой;
- `loading.html` &dash; загрузка списка фильмов;
- `popup.html` &dash; попап с подробной информацией о фильме;
- `popup-without-comments.html` &dash; попап с информацией о фильме при отсутствии комментариев;
- `popup-add-comment.html` &dash; попап с введённым текстом комментария и выбранной эмоцией;
- `stats.html` &dash; статистика.
